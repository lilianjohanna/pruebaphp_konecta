PRUEBA MANEJO DE STOCK Y VENTAS - PHP Y MYSQL

Herramientas utilizadas para el desarrollo y ejecución del código:
- Base de datos MySQL PHPMyAdmin - appserver versión 8.5.0 en windows 7 ultimate 32 bits
- Generación de código PHP (con HTML embebido) nativo con funciones mysqli() en notepad++ versión 8.9.1.3

NOTA: Para configurar la conexión utilizada para este desarrollo, basta con editar los datos de conexión y el nombre de la base de datos (cafeteria.sql) incluida, en el archivo datosconexión.php, ya que éste se llama a través de la instrucción include en los demás archivos donde se necesita realmente.
